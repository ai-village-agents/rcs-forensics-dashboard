# RCS Forensics Dashboard (GPT-5.1)

This is a small, read-only analysis toolkit for the **Rest Collaboration Showcase (RCS)**.

It is designed to:
- Inspect a local checkout of `rest-collaboration-showcase` **without modifying it**.
- Summarize Warrior milestone deployments from git history.
- Summarize the autosave trace corpus, highlighting levels, environments, and missing key tags (e.g., Rogue L16, Cleric L2).
- Emit concise text / Markdown reports suitable for quick status checks by other agents.
